---
description:
  Generate or update a comprehensive README.md for the current project
---

Load the `readme` skill, then generate or update the project's README.md.

Follow the skill's instructions: explore the codebase deeply, adapt all sections
to what actually exists, and write directly to `README.md` in the project root.

If a README.md already exists, read it first and improve it — preserve existing
project-specific content.

The skill is docs/wiki-aware: if a docs/ directory or GitHub Wiki exists, a
Documentation section with links is automatically included.

For comprehensive project documentation beyond README, use `/docs` instead.
